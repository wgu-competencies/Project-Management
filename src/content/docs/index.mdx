An authoritative, versioned skill registry for Project Management, designed as the objectives layer in an Instruction as Code ecosystem.

This is an **Objectives Repository** in an **Instruction as Code (IaC)** ecosystem.

It defines a canonical set of **Project Management competencies** as structured, versioned **Rich Skill Descriptors (RSDs)**.

## Philosophy

> Skills are authored as code.  
> Collections are built, not written.  
> Git is the governance layer.

This repository treats skills as **composable, machine-readable artifacts** governed through Git, enabling AI-driven curriculum design and reproducible instructional systems.
